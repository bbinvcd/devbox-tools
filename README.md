# DevBox - Developer Tools (Chrome Extension)

這是一個為開發者設計的多功能工具箱（All-in-one developer toolbox），包含**圖片格式轉換 (Image Converter)**、**批次重新命名 (Renamer)**、**SVG 優化 (SVG Optimizer)** 等多種實用工具。

本專案儲存庫（Repository）只包含已經建置（Build）完成的 Chrome 擴充功能檔案，您可以直接下載並手動安裝至 Chrome 瀏覽器中。

---

## 🚀 快速安裝教學

請按照以下步驟下載並安裝此 Chrome 擴充功能：

### 第一步：下載專案 ZIP 檔
1. 前往 GitHub 專案頁面：[devbox-tools](https://github.com/bbinvcd/devbox-tools.git)
2. 點擊右上角的綠色 **「Code」** 按鈕。
3. 在下拉選單中點擊 **「Download ZIP」**。
4. 下載完成後，將 ZIP 壓縮包解壓縮至您的電腦中（例如解壓至桌面或下載資料夾）。

> [!NOTE]
> 解壓縮後，您會看到一個名為 `devBox-tools` 的資料夾，裡面包含 `manifest.json`、`index.html` 以及 `background.js` 等核心檔案。

---

### 第二步：開啟 Chrome 擴充功能開發者模式
1. 開啟您的 **Chrome 瀏覽器**。
2. 在網址列輸入 `chrome://extensions/` 並按下 Enter，或者點擊瀏覽器右上角的「選單 (三個點)」 > 「延伸功能」 > 「管理延伸功能」。
3. 在擴充功能頁面的右上角，將 **「開發者模式」 (Developer mode)** 切換為**開啟**狀態。

---

### 第三步：載入擴充功能
1. 點擊頁面左上角的 **「載入未封裝項目」 (Load unpacked)** 按鈕。
2. 在彈出的檔案選擇視窗中，選擇您剛剛解壓縮得到的 **`devBox-tools`** 資料夾。
3. 點擊「選擇資料夾」或「確定」。

🎉 **恭喜您！安裝完成！** 
您現在可以在 Chrome 擴充功能清單中看到 **DevBox - Developer Tools**，並開始使用這些開發者工具了！
